| Syntax    | Functionality                                      |
| --------- | -------------------------------------------------- |
| .         | Any character                                      |
| ^         | Start of line                                      |
| $         | End of line                                        |
| \\<       | start of word                                      |
| \\>       | End of word                                        |
| \|        | logical or                                         |
| +         | one or more                                        |
| ?         | zero or  more                                      |
| ()        | group characters                                   |
| \[\]      | Character set                                      |
| \[^\]     | Negated character set                              |
| \[a - z\] | all lower case characters                          |
| \[0 - 9\] | all digits                                         |
| \\1       | backreference an earlier capture group (1-indexed) |
